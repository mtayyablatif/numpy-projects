# NumPy Projects L1

A set of small Python + NumPy projects built while learning core array operations — array math, boolean masking, argmax/argmin, and vectorized calculations.

## Projects

### 1. Grade Tracker (`grade_tracker.py`)
Takes subject-wise marks as input and prints a formatted report card with average, best/worst subject, and pass/fail status per subject.

### 2. Budget Analyzer (`budget_analyzer.py`)
Takes monthly expense categories and amounts as input, calculates total spend, percentage breakdown per category, and estimated savings from cutting the top expense.

### 3. Dice Roll Probability Checker (`diceroll_probability_checker.py`)
Simulates thousands of dice rolls and shows how the probability of rolling a 6 converges toward the true 1/6 as trial size increases — a live demo of the Law of Large Numbers.

Each script prompts for input in the terminal.

## Concepts practiced
- NumPy array creation and vectorized operations
- `np.mean()`, `np.sum()`, `np.max()`, `np.min()`, `np.argmax()`, `np.argmin()`
- Boolean masking and `np.random.choice`
- f-string formatting, `zip()`, and Python `input()` handling

## Tech Stack
Python 3, NumPy
